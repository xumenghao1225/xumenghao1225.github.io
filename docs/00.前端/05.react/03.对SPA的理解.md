# SPA

1. 单页web应用（single page web application, SPA）.
2. 整个应用中只有`一个完整的页面`；
3. 点击页面中的链接`不会刷新`页面，只会做页面的`局部刷新`；
4. 数据都需要通过ajax请求获取，并在前端异步呈现出来。
